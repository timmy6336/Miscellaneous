A simple api for the Hatchways Assesment.
this was made using javascript/node.
it uses express, node-fetch, request, memory-cache dependices.
for testing it uses jest and supertest dev dependicies.
use 'npm i' to install all non dev dependices
use 'npm i supertest --save-dev' and 'npm i jest --save-dev' to instal dev dependices
use 'npm start' to start
use 'npm test' to run unit tests
this app makes use of port 3000.
